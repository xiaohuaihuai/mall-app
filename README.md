#### 介绍
vue+vant的移动端商城

#### 软件架构
软件架构说明


#### 安装教程

```bash
# 克隆项目
git clone https://github.com/xiaohuaihuai/mall-app.git

# 进入项目目录
cd mall-app

# 安装依赖
npm install
```

#### 本地调试

```bash
# 启动服务
npm run serve
```

1.  浏览器访问 http://localhost:9001
2.  浏览器F12设置手机模式

#### 本地打包并预览

```bash
# 本地打包
npm run buildDev

# 进入dist目录
cd dist
```

1.  使用浏览器打开index.html
2.  浏览器F12设置手机模式

#### 发布

```bash
# 构建生产环境
npm run build
```
